== Description ==

Burnman's Subjot Button plugin places a button at the end of your posts, allowing visitors to share it on Subjot.com easily.

Originally got the idea from @lambdabass on Subjot, who may have been the first person to add a Subjot button to a blog.


== Installation ==

1. Unzip and Upload the folder 'burnmans-subjot-button' to the '/wp-content/plugins/' directory
2. Activate the plugin through the 'Plugins' menu in WordPress
